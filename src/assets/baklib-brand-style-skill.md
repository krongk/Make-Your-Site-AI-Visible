# Baklib 品牌视觉风格

## 风格描述

Baklib 是面向企业的知识管理 SaaS 平台，视觉风格体现：

- **专业可信**: 干净、现代的商务感
- **技术友好**: 适度的科技感，但不冰冷
- **易于理解**: 清晰的视觉层次，直观的图标系统

整体风格定位：Modern B2B SaaS, clean and professional, tech-enabled but human-friendly

## 渲染风格（已确定）

Baklib 营销配图统一采用 **Liquid Glass 液态玻璃渲染（Liquid Glass stylized render）**：

- **材质语言**: 半透明磨砂玻璃 + 清亮凝胶感体积；柔和折射、轻微色散、细腻焦散（caustics），边缘圆润
- **形体语言**: 仍是 **风格化角色/图标**（非写实人像），通过体积与高光表达结构，而不是黑色硬描边
- **光照**: 柔和三点布光 + 环境反射；高光干净克制，阴影为柔和渐变
- **UI/图标**: 与角色同一套 Liquid Glass 材质与高光规则，圆角一致、反射一致

**绝不使用**：2D 硬边扁平矢量、低多边形塑料玩具感、写实人像照片、日系动漫赛璐璐、过度噪点/胶片颗粒

| 维度 | 规范 |
|------|------|
| 描边 | **无硬黑描边**；用折射、厚度差与高光带表达轮廓 |
| 表面 | 磨砂玻璃 + 凝胶体积；干净、通透、少污渍 |
| 光照 | 柔和主光偏暖，补光柔和，反射层次清晰但不花 |
| 阴影 | 柔和渐变阴影，边缘过渡自然 |
| 高光 | 细长柔和高光带 + 少量 specular sparkle（避免金属镜面人像） |
| 参考 | Apple / VisionOS 式玻璃 UI、现代 SaaS 3D 吉祥物（但保持 Baklib 角色造型） |

## 字体规范

- **主字体**: Open Sans, SF Pro, 或类似的无衬线字体（Sans-serif）
- **标题**: Bold, 24-48pt, 颜色 `#1E293B`
- **正文**: Regular, 14-18pt, 颜色 `#1E293B` 或 `#64748B`
- **标签**: Medium, 12-14pt, 颜色 `#64748B`
- **原则**: 中英文混排时，英文使用半角，中文使用全角，保持视觉平衡

## 视觉元素

- **圆角**: 统一圆角体系（图标/UI/玻璃面板一致）
- **厚度**: 玻璃元素要有“厚度感”（边缘明暗变化 + 轻微折射）
- **投影**: 柔和环境遮蔽（ambient occlusion）+ 轻量接触阴影，避免脏灰
- **线条**: 连接线用柔和发光曲线（偏青绿色），避免硬折线图标语言
- **图标**: Liquid Glass 材质一致；颜色严格落在品牌色板内

## 构图原则

- **留白**: 充足的留白，避免拥挤
- **对齐**: 严格的对齐系统，使用网格（8px grid）
- **层次**: 前景（角色/图标）- 中景（界面/内容）- 背景（渐变/纹理）
- **焦点**: 每图一个视觉焦点，使用颜色对比引导视线

## 场景应用

### 痛点场景（分散管理）

- **背景**: `#334155` 深色，有柔和的深度感
- **氛围**: 混乱、压抑，但仍保持“干净的高级感”（玻璃碎裂感用轻量粒子/折射光表达，避免脏噪点）
- **图标**: 轻微错位/漂浮/断开连接，材质仍统一
- **角色**: 困惑表情；侧光略戏剧但仍柔和

### 解决方案场景（聚合统一）

- **背景**: `#F8FAFC` 浅色，有微妙渐变（白到浅青绿 `#E0F2F1`）
- **氛围**: 清晰、有序、高效
- **图标**: 对齐、连接、流向中心；连接线可带柔和发光
- **角色**: 自信表情；整体光照更均匀

### 过渡场景（转变过程）

- **背景**: 从左到右渐变（深 `#334155` 到浅 `#F8FAFC`）
- **氛围**: 动态、流动、转变（用折射/光流表达，而不是贴纸特效）

## 与其他风格的区分

- **vs 消费级互联网**: 更专业、更克制，减少花哨的装饰
- **vs 传统企业软件**: 更现代、更友好，避免过时的渐变和阴影
- **vs 创意机构**: 更结构化、更一致，强调功能性而非艺术性

## 一致性检查清单

生成图片前，确认以下品牌要素：

- [ ] 主色使用 `#0D9488`（青绿色）
- [ ] 强调色使用 `#ea580c`（橙色）
- [ ] 渲染风格为 **Liquid Glass 液态玻璃**（风格化，不是写实）
- [ ] 无硬黑描边；轮廓靠厚度/折射/高光表达
- [ ] 角色/图标/UI 的 **玻璃材质一致**（磨砂程度、折射强度、高光形状）
- [ ] 字体为无衬线（sans-serif）
- [ ] 角色头身比约 **1:4 ~ 1:4.5**（与角色档案一致）
